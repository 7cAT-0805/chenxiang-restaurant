# 沉香鍋燒 (Chen Xiang Guo Shao) 官方網站

這是一個為「沉香鍋燒」設計的傳統風格官方網站。設計靈感源自台南傳統鍋燒意麵的優雅與質樸。

## 功能特點
- **典雅設計**：採用米色與深灰色的和諧配色，搭配書法字體（馬善政體）呈現傳統美感。
- **多頁式架構**：首頁、關於我們、美味菜單、人才招募四個獨立頁面。
- **響應式佈局**：支援手機、平板與電腦瀏覽。
- **現代動畫效果**：平滑過渡、懸停動畫、淡入效果等視覺增強。
- **精緻視覺**：陰影層次、漸變背景、置頂導航欄等專業設計元素。

## 頁面結構
- **index.html**：首頁，展示主視覺與品牌理念
- **about.html**：關於我們頁面
- **menu.html**：美味菜單頁面（展示菜單圖片）
- **recruitment.html**：人才招募頁面

## 如何開啟
直接在瀏覽器中開啟 `index.html` 即可預覽網站。

## 技術棧
- HTML5
- CSS3 (Flexbox, Grid, Animations, Gradients)
- JavaScript (Vanilla JS)
- Google Fonts (Ma Shan Zheng, Noto Serif TC)

## 設計特色
- 置頂式半透明導航欄
- 漸層背景與陰影層次

- 懸停互動效果
- 波紋按鈕動畫
